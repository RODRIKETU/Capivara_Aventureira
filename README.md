# Capivara Aventureira 🌿🎒

![Android Version](https://img.shields.io/badge/Android-5.0%2B-brightgreen.svg)
![Build Status](https://img.shields.io/badge/Build-Passing-brightgreen.svg)

**Capivara Aventureira** é um aplicativo Android focado em oferecer uma experiência web integrada e fluida. Originalmente derivado do projeto Camara Cidadã, esta versão foi personalizada para atender às necessidades específicas da plataforma [capivara.aporttec.com](https://capivara.aporttec.com).

## ✨ Principais Funcionalidades

- **Navegação Web Integrada**: WebView otimizado para carregamento rápido e navegação suave.
- **Animações de Carregamento**: Transições visuais agradáveis enquanto os dados são processados.
- **Gestão de Conectividade**: Detecção automática de estado de rede com tela de erro amigável.
- **Suporte Edge-to-Edge**: Aproveitamento total da tela em dispositivos modernos.
- **Ampla Compatibilidade**: Suporte desde o Android 5.0 (API 21) até as versões mais recentes.

## 🚀 Como Começar

### Pré-requisitos
- Android Studio Ladybug ou superior.
- JDK 11+.
- SDK do Android (API 21+).

### Instalação
1. Clone o repositório:
   ```bash
   git clone https://github.com/RODRIKETU/Capivara_Aventureira.git
   ```
2. Abra o projeto no Android Studio.
3. Sincronize o Gradle.
4. Execute o aplicativo em um emulador ou dispositivo físico.

## 🛠️ Tecnologias Utilizadas

- **Kotlin**: Linguagem principal de desenvolvimento.
- **ViewBinding**: Para interação segura com componentes de UI.
- **Glide**: Carregamento eficiente de imagens.
- **WebView**: Motor de renderização web nativo do Android.
- **Material 3**: Design moderno e responsivo.
